1、首先要进Accelrys_Materials_Studio_8.0_Win找到autorun.exe，右键管理员方式autorun可执行文件。可一直默认，运行到最后finish界面时，可以点确定，
会弹出一个注册窗口，选择最下面的 "For more options, start the License Administrator"。
#注意（安装时候直接照上面安装就可以，下面是作者的建议，按照下面也行）
安装过程中考虑注册的权限问题，我把那个勾去掉Configure Materials Studio 8.0 Licensing，然后finish，
再从开始>所有程序>Accelrys>License>右键管理员方式打开Configure Materials Studio 8.0 Licensing，我要取得足够的权限，防止注册失败！
然后会弹出一个注册窗口，选择最下面的 "For more options, start the License Administrator"。点next，暂停一下，完成第二步，继续。
2、解压crack文件，找到msi.lic文件，右键用记事本打开或打不开改成txt格式再修改，修改完改回去就行
把第一行中的this_host替换为你的计算机名（点计算机，右键属性查找。特别注意计算机名不能为中文名的，必须是英文或数字），保存文件。
然后把这个msi.lic文件拷贝到 C:\Program Files (x86)\Accelrys\LicensePack\Licenses\ or C:\Program Files\Accelrys\LicensePack\Licenses\中去。
3、继续第一步，License file > Install License > Browse (browse to "msi.lic" > Install > OK，其中msi.lic文件在刚才拷贝的文件夹中。
4、完美激活！在安装路径里面找到Materials Studio 8.0\bin里面找MatStudio.exe发送快捷方式到桌面
